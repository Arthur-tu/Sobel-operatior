# Sobel-operatior
Фильтр Собеля – это оператор, вычисляющий приблизительный градиент яркости, используя градиенты изображений вдоль осей x и y. Наиболее распространенным примером практического использования является определение границ объектов на изображении, т.е. точек резкого изменения яркости. 
Оператор использует ядра 3x3, с которыми и происходит свертка рассматриваемого изображения по вертикали и по горизонтали. Пусть A – исходное изображение, представленное матрицей, элементами которой являются пиксели изображения, а G_x и G_y – два изображения, на которых каждая точка содержит приближённые производные по x и по y.
